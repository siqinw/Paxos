To run TCP server:
cd src/
java TCP.TCPKeyValueStoreServer <port>

To run TCP client:
cd src/
java TCP.TCPKeyValueStoreClient <server_address> <port>

To run UDP Server:
cd src/
java UDP.UDPKeyValueStoreServer <port>

To run UDP Client:
cd src/
java UDP.UDPKeyValueStoreClient <server_address> <port>
